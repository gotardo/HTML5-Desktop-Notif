HTML5 Desktop Notifications

About this library

This class provides a way to easily launch HTML5 Notifications taking advantage of the API Specification at Chromium Projects.

Use

			
//Some lines to configure the behavior

Notif.config({
	notificationType: "text",
	icon: "css/img/Alf-48.png",
	autoclose: 3,
	fallback: function (){
	/* 
		Your code for browsers which 
		doesn't support Webkit Notifications.
	*/
	}
});

//Show notification	

Notif.show({ 
	title: "Warning!", 
	msg: "You have a new notification."
});
